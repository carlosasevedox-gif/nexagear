# NexaGear 🛒⚡

Réplica em **HTML e CSS puro** (sem JavaScript) do site NexaGear, criado originalmente no Figma.

> Acessórios de tecnologia para estudo, trabalho e organização.

## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Página inicial — hero, ofertas da semana, frete grátis e diferenciais |
| `produtos.html` | Catálogo com categorias e grade de produtos |
| `produto.html` | Detalhe do produto (Mochila TechBag) |
| `quem-somos.html` | Sobre a empresa — missão, visão e valores |
| `contato.html` | Formulário de contato + informações |
| `cadastro.html` | Criação de conta |
| `pedidos.html` | Carrinho / resumo do pedido |
| `busca.html` | Busca com filtros |

## Como visualizar

Basta abrir o `index.html` em qualquer navegador — não há dependências nem build.

## Tecnologias

- HTML5 semântico
- CSS3 (Grid, Flexbox, variáveis CSS)
- Fonte [Poppins](https://fonts.google.com/specimen/Poppins) via Google Fonts
- Imagens ilustrativas via [Unsplash](https://unsplash.com)
- Ícones SVG inline (sem bibliotecas)
